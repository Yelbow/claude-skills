description: Interview the user relentlessly about a plan, proposal, or design until reaching shared understanding, resolving each branch of the decision tree one by one. Use when user wants to stress-test a plan, get grilled on a design, or mentions "grill me". Works for code, content, strategy, and client proposals.

Interview me relentlessly about every aspect of this plan until we reach a shared understanding. Walk down each branch of the design tree, resolving dependencies between decisions one-by-one. For each question, provide your recommended answer.

Ask the questions one at a time.

If a question can be answered by checking existing context (skills, context files, earlier messages in this conversation, or the GitHub repo) instead of asking, do that.
